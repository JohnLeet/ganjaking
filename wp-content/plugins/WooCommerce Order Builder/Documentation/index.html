    The Back-End Interface

    WWOB’s back-end interface is built carefully with consideration to simplicity and easiness. After installing the plugin you’ll be introduced to a new metabox in the product edit page called “WPAli: Woocommerce Order Builder”, here you can easily add unlimited number of products/Items or sub-items of your product. Additionally, the interface gives you the ability to restrict items selection by front-end user to a specific number and even set a minimum number of selections, meaning you can accept only a specific number of items to be bought within a single product. Not only that, but there are also extra options available you to use if needed. Moreover, you can at any time enable or disable the the plugin’s functionality on product page, that means you can switch between WooCommerce default front-end interface and WWOB’s with a single click.

    Installation - #back to top

    Get a Installation Service Ask a Question

    WooCommerce First

    If you have an existing WordPress site and want to use WWOB, you’ll need to have WooCommerce installed first.

    To install WooCommerce:

    1. Log into your WordPress site.
    2. Go to: Plugins > Add New.
    3. Search for ‘WooCommerce’.
    4. Select Install Now when you see it’s by Automattic.
    5. Select Activate Now and configure your store using WooCommerce Wizard!

    WWOB Installation

    1. Download the .zip file provided to you at the time of purchase
    2. Go to: WordPress Admin > Plugins > Add New to upload the file you downloaded ('wpali-woocommerce-order-builder.zip').
    3. Activate the plugin.

    Once installed and activated, WWOB settings menu will appear under products tabs in your dashboard.

    More information at: Installing and Managing Plugins.

    Creating a product - #back to top

    Get a Set-Up or Customization Service

    To put WWOB to action, you need to create a product.

    1. Go to: Products > Add Product. A familiar WooCommerce interface for product creation appears.

    2. Enter a title and description for your product.

    3. You should have "Simple Product" selected by default from Product Data dropdown menu, now enter the value 0 in Regular price field to allow WWOB's dynamic pricing.

    (If you make regular price more than 0, the price you entered will be added to each order, this is helpful if you have a static price with specific number of item included, in this case you can define your price here, and set items price later to 0)

    4. Scroll to "WPAli: Woocommerce Order Builder" metabox, check Yes from Enable Order Builder option to to enable the builder for this product.

    5. In the product row, give your item a name, description(optional). After that, you can start adding sub-items to this product using the repeatable field labeled sub-items, and the plus button below it.

    Note: Only label, and sub-items fields are required to create a product, the rest are optional.

    6. After you finish adding the sub-items, click publish.

    The result should be the similar to this (One section with a title, description( if added ), items, and a price calculator on the right):

    Max select: The maximum number of items that the customer can select per product (leave empty to set to unlimited). Here is an example where max select is set to value (2):

    FAQ #back to top

    Does it have the ability to add “special instructions” by customer?

    There is an additional feature in the product edit page, in Extra Options tab exactly where you can allow your customer to submit special instructions with his order.

    Can it be used as a product basket?

    Yes, you can create product basket with WWOB and restrict the customer to a specific number of items and force a minimum of items as well. If your customer have to choose exactly 5 items to complete an order, you can use the value (5) in Min Select and Max Select fields in the product select page for the specific product you want to apply this to.

    Is it possible to combine products but also allow customer to choose size?

    Yes, this can be easily done by enabling product options from the product edit page. Go to WPAli: WooCommerce Order Builder > Extra Options > check Enable Product Options and select the options you want to add, in this case Size options.
