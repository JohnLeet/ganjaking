== Description ==

This WooCommerce extension gets you set up to sell photos in the blink of an eye and makes creating products your fans and clients can buy as simple as dragging & dropping images!
Visit our [product page](https://woocommerce.com/products/woocommerce-photography/) for more info.

== Minimum Requirements ==

* PHP version 5.4 (PHP 7.4 or greater is recommended)
* MySQL version 5.0 (MySQL 5.7 or greater is recommended)

== Installation ==

1. Unzip and upload the plugin’s folder to your /wp-content/plugins/ directory.
2. Activate the extension through the ‘Plugins’ menu in WordPress.
3. Go to WooCommerce > Settings > Shipping & Delivery to configure the plugin.
